# AI秘書テンプレート

**話しかけるだけで、あなた専用のAI秘書が動き出します。**

---

## 準備するもの

| 必要なもの | 説明 | 費用 |
|---|---|---|
| **Claude Code** | Anthropicが提供するAIツール | 無料でダウンロードできます |
| **Claude Pro** | Anthropicの有料プラン | 月額$20（約3,000円）から |
| **パソコン** | Mac・Windowsのどちらでも使えます | — |

---

## セットアップ手順

### ステップ1: Claude Codeをダウンロードする

1. **https://claude.ai/download** をブラウザで開いてください
2. 画面に「Download for Windows」または「Download for Mac」というボタンが表示されます
3. あなたのパソコンに合った方のボタンをクリックしてください
4. ダウンロードが始まります（数分かかる場合があります）

---

### ステップ2: Claude Codeをインストールする

**Windowsの方:**
1. ダウンロードされたファイル（`.exe`という拡張子のファイル）をダブルクリックします
2. 「このアプリがデバイスに変更を加えることを許可しますか？」と聞かれたら「はい」をクリックします
3. 「Next」（次へ）というボタンを何度かクリックします（設定はそのままでOKです）
4. 「Install」または「Finish」をクリックしてインストール完了です

**Macの方:**
1. ダウンロードされたファイル（`.dmg`という拡張子のファイル）をダブルクリックします
2. 画面が開いたら、アイコンを「Applications」フォルダにドラッグ＆ドロップします
3. インストール完了です

---

### ステップ3: Gitをインストールする（Windowsの方のみ）

Windowsの方はもう1つ「Git」というツールが必要です。Macの方はスキップしてください。

1. **https://git-scm.com/downloads/win** をブラウザで開いてください
2. 「Click here to download」をクリックしてダウンロードします
3. ダウンロードされたファイルをダブルクリックします
4. 「Next」（次へ）ボタンを何度かクリックします（設定はそのままでOKです）
5. 「Install」をクリックしてインストール完了です

> **「Gitって何？」と思った方へ**
> GitはAI秘書をパソコンにダウンロードするために必要な裏側のツールです。
> 使い方を覚える必要はありません。インストールするだけでOKです。

**Macの方へ:**
Macにはgitが最初から入っている場合がほとんどです。
初回だけ「Xcode Command Line Toolsをインストールしますか？」というポップアップが出る場合があります。
その場合は「インストール」をクリックするだけでOKです。特別な操作は必要ありません。

---

### ステップ4: Claude Codeを起動してサインインする

1. インストールしたClaude Codeを開きます
   - Windowsの方: デスクトップまたはスタートメニューから「Claude」を探してクリック
   - Macの方: アプリケーションフォルダから「Claude」をクリック
2. 初回起動時はサインイン画面が表示されます
3. Anthropicのアカウントでサインインしてください
   - アカウントがない方は「Sign up」から作成できます（メールアドレスとパスワードでOK）

---

### ステップ5: AI秘書をセットアップする

Claude Codeが起動してチャット画面が表示されたら、いよいよAI秘書のセットアップです。

**下の文字をそのままコピーして、チャット画面の入力欄に貼り付けてEnterキーを押してください。**

```
セットアップして。
https://github.com/joshicrea/joshicrea-secretary.git を
ホームフォルダ直下に ai-secretary という名前でcloneして
AI秘書のオンボーディングを始めてください。
```

> **「ホームフォルダ」って何？と思った方へ**
> どのフォルダで起動してもAI秘書が動くように、決まった場所に保存します。
> AI秘書が自動で場所を決めてくれるので、難しく考えなくて大丈夫です。

> **コピーの仕方**
> 上の文字を選択して、右クリック→「コピー」または Ctrl+C（Macは⌘+C）でコピーできます。
> 貼り付けは入力欄をクリックしてから Ctrl+V（Macは⌘+V）です。

貼り付けてEnterを押すと、AI秘書がいくつか質問してきます。
**答えるほど、あなたに合った動き方になります。**

---

### ステップ6: 質問に答える

AI秘書が以下のような質問をしてきます。1つずつ答えてください。

- お名前（どう呼んでほしいか）
- どんなお仕事をしているか
- 今一番困っていること・時間がかかっていること
- ホームページやSNSのURL（あれば）
- GmailやGoogle Calendarと連携するか

> **URLや資料がある方へ**
> `resources` というフォルダに名刺・パンフレット・LPなどを入れておくと、
> AI秘書があなたの事業をより深く理解できます。

全ての質問に答えると、あなた専用のAI秘書が完成します。

## セットアップ後の使い方

セットアップが完了したら、普通に話しかけるだけで動きます。

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「今日何する？」 | 今日のタスク・予定・支払いをまとめて整理します |
| 「おはよう」 | メール・予定・タスクを一括確認します |
| 「メール見て」 | 重要なメールだけ整理して教えます |
| 「〇〇をタスクに追加して」 | やることを記録します |
| 「タスク見せて」 | やり残していることを一覧表示します |
| 「〇〇さんを登録して」 | お客さんの情報を記録します |
| 「〇〇円使った」 | 経費を記録します |
| 「見積書を作って」 | 下書きを生成します |
| 「今月何やった？」 | 月次まとめを生成します |
| 「何ができる？」 | 全機能の一覧を表示します |

**正確な言葉を使わなくて大丈夫です。**
「ちょっとメール見てもらえますか」「予定どうなってたっけ」でも動きます。

## うまくいかないときは

**Claude Codeに「うまくいかない」と話しかけてください。**
日本語で状況を確認して案内します。

よくあるトラブル:

| 症状 | 対処法 |
|---|---|
| エラーメッセージが出た | 「エラーが出た」とAI秘書に話しかける |
| セットアップが途中で止まった | 「セットアップを続けて」と話しかける |
| Gitのインストールを求められた | ステップ3を確認してインストールする |
| ログインを求められた | Claude Proのアカウントでサインインする |

## よくある質問

**Q. GmailやGoogle Calendarと連携しないと使えませんか？**
A. いいえ。連携なしでも、タスク管理・お客さん管理・経費メモ・見積書作成など多くの機能が使えます。メールの確認は連携後に使えます。

**Q. Macでも使えますか？**
A. はい。Mac・Windowsどちらでも動作します。

**Q. パソコンが苦手でも大丈夫ですか？**
A. 大丈夫です。セットアップ中に詰まっても、AI秘書に「どうすればいい？」と聞けば日本語で案内します。

**Q. 入力した情報は外に出ますか？**
A. いいえ。入力した情報はあなたのパソコンの中にあります。外部には送信されません。

**Q. メールを勝手に送られたりしますか？**
A. しません。メールの送信・予定の変更は必ずAI秘書が確認を取ってから実行します。

## 資料をお持ちの方へ

名刺・LP・パンフレット・会社案内などがあると、AI秘書があなたの事業をより深く理解できます。

`resources` フォルダに入れてから「資料を入れました」と話しかけてください。
AI秘書が自動で読み取ります。
