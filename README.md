# AI秘書テンプレート

**話しかけるだけで、あなた専用のAI秘書が動き出します。**

---

## 必要なもの

| 必要なもの | 説明 |
|---|---|
| **Claude Code** | Anthropicの開発ツール（インストール方法は付属の動画で解説） |
| **Claude Pro** | Anthropicの有料プラン（月額$20から） |
| **パソコン** | Mac・Windows・Linuxのいずれか |

---

## セットアップ手順

### ステップ1: Claude Codeを開く

インストール済みのClaude Codeを起動してください。

### ステップ2: コピペ1行で開始

チャット欄に以下をそのままコピペして送信してください:

```
セットアップして。
https://github.com/joshicrea/joshicrea-secretary.git をcloneして
AI秘書のオンボーディングを始めてください。
```

あとはAI秘書の質問に答えるだけです。

---

## セットアップで何が起きるか

1. **AI秘書がヒアリングを始めます**
   - お名前・事業内容・困っていることを聞かれます
   - HPやSNSのURLを入力するとAIが自動でリサーチします
   - 名刺・LP等の資料を入れると理解の精度が上がります

2. **あなた専用の設定が自動で作られます**
   - 事業内容・使用ツール・文体をAIが記憶します
   - 毎回説明しなくても、あなたのことを理解した状態で動きます
   - 使えば使うほど精度が上がります（memory/フォルダに学習が蓄積）

3. **話しかけるだけで動きます**

---

## 使い方一覧

| 言葉 | 何をしてくれるか |
|---|---|
| 「おはよう」 | メール・予定・タスクを一括確認してまとめます |
| 「メール見て」 | 重要メールだけ整理して報告します |
| 「予定確認して」 | 今日・今週の予定を確認します |
| 「タスク追加して〇〇」 | タスクを記録して期限を管理します |
| 「状況教えて」 | 未完了タスクの一覧を確認します |
| 「振り返り」 | 今週の振り返りと来週の優先事項を整理します |
| 「〇〇スキル作って」 | あなたの仕事に合った新しい機能を追加します |

---

## 育てる仕組み

このAI秘書は使えば使うほど賢くなります。

- 学習内容が `memory/` フォルダに蓄積されます
- 修正指示を出すと次回から改善されます
- 「〇〇スキル作って」で新しい機能を随時追加できます
- Obsidianと連携すると学習内容をVaultに記録できます

---

## 資料を持っている方へ

`resources/` フォルダに名刺・LP・パンフレット等を入れてください。
資料が多いほど、AI秘書の理解精度が上がります。

詳しくは [resources/README.md](resources/README.md) をご覧ください。

## よくある質問

**Q. Gmail・Google Calendarと連携しないと使えませんか？**
A. いいえ。タスク管理・スキル追加は連携なしで即日使えます。メール・カレンダー確認は連携後に使えます。

**Q. Macでも使えますか？**
A. はい。Mac・Windows・Linuxで動作します。

**Q. うまく動かない場合は？**
A. AI秘書に「うまく動かない」と話しかけてください。状況を確認して対処します。

---

## 注意事項

- AIは間違えることがあります。重要な判断は必ずご自身で確認してください
- メールの送信・予定の変更は必ずAIが確認を取ってから実行します
- `resources/` に入れた資料はあなたのパソコンの中にあります（外部送信されません）
