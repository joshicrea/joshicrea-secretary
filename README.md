# AI秘書テンプレート

**話しかけるだけで、あなた専用のAI秘書が動き出します。**

---

## 準備するもの

| 必要なもの | 説明 | 費用 |
|---|---|---|
| **Claude Code** | Anthropicが提供するAIツール | 無料でダウンロードできます |
| **Claude Pro** | Anthropicの有料プラン | 月額$20（約3,000円）から |
| **パソコン** | Mac・Windowsのどちらでも使えます | — |

---

## セットアップ手順

### ステップ1: Claude Codeをダウンロードしてインストールする

1. **https://claude.ai/download** をブラウザで開いてください
2. 「Download for Windows」または「Download for Mac」をクリックしてダウンロードします

**Windowsの方:**
- ダウンロードされた `.exe` ファイルをダブルクリックして「Next」を進めていくとインストール完了です

**Macの方:**
- ダウンロードされた `.dmg` ファイルを開き、アイコンを「Applications」フォルダにドラッグ＆ドロップします

---

### ステップ2: Claude Codeを起動してサインインする

1. インストールしたClaude Codeを開きます
2. Anthropicのアカウントでサインインしてください
   - アカウントがない方は「Sign up」から作成できます（メールアドレスとパスワードでOK）

---

### ステップ3: AI秘書プラグインをインストールする

お使いのパソコンに合わせて手順が異なります。

---

**Windowsの方:**

Claude Codeを開いて、以下の文章を**そのままコピーして貼り付け**、Enterを押してください。

```
以下のURLからAI秘書プラグインのインストールスクリプトを取得して、内容を確認してから実行してください: https://raw.githubusercontent.com/joshicrea/joshicrea-secretary/master/install.ps1
```

Claude Codeが内容を確認したうえで自動でインストールを実行します。

---

**Macの方:**

Macの「**ターミナル**」アプリを開いてください。
（開き方: Launchpadで「ターミナル」と検索、または `アプリケーション → ユーティリティ → ターミナル`）

ターミナルが開いたら、以下の1行を**そのままコピーして貼り付け**、Enterを押してください。

```
curl -fsSL https://raw.githubusercontent.com/joshicrea/joshicrea-secretary/master/install.py | python3
```

「インストール完了！」と表示されたら成功です。

> **なぜMacはターミナルなのか？**
> MacのClaude Codeはセキュリティ上の理由で外部スクリプトの実行を拒否します。
> ターミナルからの実行は安全な方法で、所要時間は1分以内です。

---

インストールが完了したら、**Claude Codeをいったん閉じて、もう一度開いてください。**

---

### ステップ4: AI秘書のセットアップを始める

Claude Codeを再起動したら、入力欄に以下を貼り付けてEnterを押してください。

```
はじめまして
```

AI秘書がいくつか質問してきます。**答えるほど、あなたに合った動き方になります。**

- お名前（どう呼んでほしいか）
- どんなお仕事をしているか
- 今一番困っていること・時間がかかっていること
- ホームページやSNSのURL（あれば）
- GmailやGoogle Calendarと連携するか

> **URLや資料がある方へ**
> `resources` というフォルダに名刺・パンフレット・LPなどを入れておくと、
> AI秘書があなたの事業をより深く理解できます。

全ての質問に答えると、あなた専用のAI秘書が完成します。

## セットアップ後の使い方

セットアップが完了したら、普通に話しかけるだけで動きます。

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「今日何する？」 | 今日のタスク・予定・支払いをまとめて整理します |
| 「おはよう」 | メール・予定・タスクを一括確認します |
| 「メール見て」 | 重要なメールだけ整理して教えます |
| 「〇〇をタスクに追加して」 | やることを記録します |
| 「タスク見せて」 | やり残していることを一覧表示します |
| 「〇〇さんを登録して」 | お客さんの情報を記録します |
| 「〇〇円使った」 | 経費を記録します |
| 「見積書を作って」 | 下書きを生成します |
| 「今月何やった？」 | 月次まとめを生成します |
| 「何ができる？」 | 全機能の一覧を表示します |

**正確な言葉を使わなくて大丈夫です。**
「ちょっとメール見てもらえますか」「予定どうなってたっけ」でも動きます。

## うまくいかないときは

**Claude Codeに「うまくいかない」と話しかけてください。**
日本語で状況を確認して案内します。

よくあるトラブル:

| 症状 | 対処法 |
|---|---|
| エラーメッセージが出た | 「エラーが出た」とAI秘書に話しかける |
| セットアップが途中で止まった | 「セットアップを続けて」と話しかける |
| プラグインが見つからないと言われた | Claude Codeを再起動してからステップ3のインストール手順をもう一度実行する |
| Macでインストールに失敗する | ターミナルで `python3 --version` と入力してPython 3が入っているか確認。入っていない場合は `xcode-select --install` を実行 |
| ログインを求められた | Claude Proのアカウントでサインインする |

## よくある質問

**Q. GmailやGoogle Calendarと連携しないと使えませんか？**
A. いいえ。連携なしでも、タスク管理・お客さん管理・経費メモ・見積書作成など多くの機能が使えます。メールの確認は連携後に使えます。

**Q. Macでも使えますか？**
A. はい。Mac・Windowsどちらでも動作します。

**Q. パソコンが苦手でも大丈夫ですか？**
A. 大丈夫です。セットアップ中に詰まっても、AI秘書に「どうすればいい？」と聞けば日本語で案内します。

**Q. 入力した情報は外に出ますか？**
A. いいえ。入力した情報はあなたのパソコンの中にあります。外部には送信されません。

**Q. メールを勝手に送られたりしますか？**
A. しません。メールの送信・予定の変更は必ずAI秘書が確認を取ってから実行します。

## 資料をお持ちの方へ

名刺・LP・パンフレット・会社案内などがあると、AI秘書があなたの事業をより深く理解できます。

`resources` フォルダに入れてから「資料を入れました」と話しかけてください。
AI秘書が自動で読み取ります。

---

## アップデート方法

AI秘書は定期的に改善されます。AI秘書に「**アップデートして**」と話しかけるだけで最新版に更新できます。

それでもうまくいかない場合は、ステップ3のインストール手順をもう一度実行してください（再インストールではなく最新版への更新になります）。

| パソコン | 手順 |
|---|---|
| Windows | Claude Codeのチャットに「以下のURLから〜」の指示を貼り付けて実行 |
| Mac | ターミナルで `curl -fsSL .../install.py \| python3` を実行 |

更新後はClaude Codeを再起動してください。
