# AI秘書テンプレート

**話しかけるだけで、あなた専用のAI秘書が動き出します。**

---

## 必要なもの

| 必要なもの | 説明 |
|---|---|
| **Claude Code** | Anthropicの開発ツール（インストール方法は購入特典の動画で解説） |
| **Claude Pro** | Anthropicの有料プラン（月額$20から） |
| **パソコン** | Mac・Windows・Linuxのいずれか |

---

## セットアップ手順

### ステップ1: Claude Codeを開く

インストール済みのClaude Codeを起動してください。
（インストールがまだの方は購入特典の動画をご覧ください）

### ステップ2: コピペ1行でセットアップ開始

チャット欄に以下をそのままコピペして送信してください:

```
セットアップして。
https://github.com/joshicrea/secretary.git をcloneして
AI秘書のオンボーディングを始めてください。
```

これだけです。あとはAI秘書の質問に答えていくだけ。

---

## セットアップで何が起きるか

1. **AI秘書がヒアリングを始めます**
   - お名前、事業内容、使っているツールを聞かれます
   - HPやSNSのURLを入力するとAIが自動でリサーチします
   - 名刺・LP等の資料があれば `resources/` フォルダに入れると精度が上がります

2. **あなた専用の設定が自動で作られます**
   - 文体・トーン・使用ツールをAIが記憶します
   - 毎回説明しなくても、あなたのことを理解した状態で動きます

3. **話しかけるだけで使えます**
   - 「メルマガ書いて」→ あなたの文体でメルマガを書きます
   - 「台本書いて」→ YouTube台本を書きます
   - 「記事書いて」→ ブログ記事を書きます
   - 「メール見て」→ 重要メールを整理します（Gmail連携時）

## 資料を持っている方へ

`resources/` フォルダに、あなたの資料を入れてください。
名刺・LP・パンフレット等、何でもOKです。
資料が多いほど、AI秘書の精度が上がります。

詳しくは [resources/README.md](resources/README.md) をご覧ください。

---

## よくある質問

**Q. Gmail・Google Calendarと連携しないと使えませんか？**
A. いいえ。連携しなくてもメルマガ・台本・記事・LINE等のコンテンツ生成機能はすぐに使えます。メール確認・カレンダー確認は連携後に使えます。

**Q. Macでも使えますか？**
A. はい。Mac・Windows・Linuxで動作します。

**Q. うまく動かない場合は？**
A. 購入者向けサポートコミュニティでご質問ください。

---

## 注意事項

- AIは間違えることがあります。重要な判断は必ずご自身で確認してください
- 金銭・契約に関わる判断はAIに任せないでください
- メールの送信・予定の変更は必ずAIが確認を取ってから実行します
- `resources/` に入れた資料はあなたのパソコンの中にあります
