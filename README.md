# AI秘書テンプレート

**話しかけるだけで、あなた専用のAI秘書が動き出します。**

---

## 必要なもの

| 必要なもの | 説明 |
|---|---|
| **Claude Code** | Anthropicが提供するAIツール（無料でダウンロードできます） |
| **Claude Pro** | Anthropicの有料プラン（月額$20から） |
| **パソコン** | Mac・Windowsのどちらでも使えます |

---

## ステップ0: Claude Codeをインストールする

まだClaude Codeをインストールしていない方は先にこちらを済ませてください。

1. **https://claude.ai/download** を開く
2. 「Download for Windows」または「Download for Mac」をクリック
3. ダウンロードされたファイルをダブルクリックしてインストール
4. インストール完了後、Claude Codeを起動するとチャット画面が開きます

---

## ステップ1: Claude Codeを開く

インストール済みのClaude Codeを起動してください。
起動するとチャット入力欄のある画面が開きます。

---

## ステップ2: 下の文字をコピーして貼り付ける

チャット画面の一番下の入力欄に、以下の文字をそのままコピーして貼り付けてください。
貼り付けたら「送信」（Enterキー）を押してください。

```
セットアップして。
https://github.com/joshicrea/joshicrea-secretary.git をcloneして
AI秘書のオンボーディングを始めてください。
```

あとはAI秘書の質問に答えるだけです。

---

## セットアップで何が起きるか

1. **AI秘書がいくつか質問してきます**
   - お名前・お仕事の内容・困っていることを聞かれます
   - ホームページやSNSのURLを入力するとAIが自動でリサーチします
   - 名刺・パンフレット等の資料を入れると精度が上がります（なくても大丈夫です）

2. **あなた専用の設定が自動で作られます**
   - お仕事の内容・使っているツール・文体をAIが記録します
   - 毎回同じことを説明しなくても、あなたのことを理解した状態で動きます
   - 使えば使うほど精度が上がります

3. **設定が終わったら、話しかけるだけで動きます**

## 使い方一覧

### 毎日の確認

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「おはよう」 | メール・今日の予定・タスクをまとめて確認します |
| 「メール見て」 | 重要なメールだけ整理して教えます |
| 「予定確認して」 | 今日・今週の予定を確認します |

### タスク・フォローアップ

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「〇〇をタスクに追加して」 | やることを記録して期限を管理します |
| 「タスク見せて」 | やり残していることの一覧を表示します |
| 「〇〇さんに3日後にフォローして」 | リマインドを記録して期限が来たら教えます |

### 顧客管理

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「〇〇さんを登録して」 | 顧客情報を記録します |
| 「〇〇さんの状況教えて」 | 最終連絡日・次のアクションを表示します |
| 「顧客一覧」 | 全顧客と連絡が必要な人を一覧表示します |
| 「〇〇さんに連絡した」 | 最終連絡日を今日に更新します |

### 経費・支払い管理

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「〇〇円使った」 | 経費を記録して今月の合計を表示します |
| 「今月の経費」 | 経費の一覧と合計を表示します |
| 「〇〇の支払い毎月〇日」 | 固定費を登録して期限前に自動通知します |

### 書類・メール作成

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「〇〇さんとアポ取りたい」 | 候補日3つ入りの返信下書きを作ります |
| 「見積書を作って」 | 金額・項目を聞いて下書きを生成します |
| 「このメールどう返せばいい？」 | メールの内容を貼り付ければ返信下書きを作ります |
| 「この返信を定型文にして」 | よく使う返信を保存して次回から提案します |

### まとめ・振り返り

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「今月何やった？」 | タスク・顧客・経費から月次サマリーを生成します |
| 「振り返り」 | 今週の振り返りと来週の優先事項を整理します |

### その他

| こう言うだけでOK | AI秘書がやること |
|---|---|
| 「後で読みたい」 | URLやメモを保存します |
| 「何ができる？」 | 使える機能の一覧を表示します |
| 「〇〇スキル作って」 | あなたの仕事に合った新しい機能を追加します |

---

## うまく話しかけられないときは

**正確な言葉を使わなくて大丈夫です。**

- 「見て」「確認して」「教えて」「お願い」など、普通に話しかけてください
- 「メールちょっと確認してもらえますか」でも動きます
- 分からないことがあれば「これどうすればいい？」と聞いてください

---

## Gmailと連携していない場合

Gmailの連携設定をしていなくても使えます。

- メールの内容をここに貼り付ければ整理・返信下書きを手伝います
- 「このメールどう返せばいい？」と言って、メールの内容をコピペしてください

---

## 育てる仕組み

このAI秘書は使えば使うほど賢くなります。

- 会話の中で知った情報を自動で記録します
- 「よく使う返信を定型文にして」と言うと次回から提案してくれます
- 「〇〇スキル作って」で新しい機能を随時追加できます

## よくある質問

**Q. GmailやGoogle Calendarと連携しないと使えませんか？**
A. いいえ。タスク管理・顧客管理・経費メモ・スキル追加は連携なしで即日使えます。

**Q. Macでも使えますか？**
A. はい。Mac・Windowsどちらでも動作します。

**Q. エラーが出て止まってしまいました**
A. AI秘書に「うまくいかない」「エラーが出た」と話しかけてください。日本語で案内します。

**Q. 何て言えばいいか分からないです**
A. 「何ができる？」と話しかけると機能一覧が表示されます。

---

## 注意事項

- AIは間違えることがあります。重要な内容は必ずご自身で確認してください
- メールの送信・予定の変更は必ずAIが確認を取ってから実行します（勝手に送信しません）
- 入力した情報はあなたのパソコンの中にあります（外部に送信されません）
