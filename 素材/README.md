# resources フォルダ

ここにあなたの事業に関する資料を入れてください。

## 入れてよいファイル

- 名刺（画像・PDF）
- サービス案内・パンフレット（PDF・画像）
- LPのスクリーンショット
- プロフィール文（テキスト・Word）
- 実績・お客様の声（テキスト・画像）
- よく使う文章・テンプレート

## 入れると何が変わるか

資料が多いほど、AI秘書があなたのことを深く理解して動けます。

- 毎回「どんな仕事ですか？」と聞かれなくなる
- あなたの文体・トーンに合った文章を書いてくれる
- ターゲット顧客に合ったコンテンツを自動で調整してくれる

## 注意

- 個人情報（パスワード・口座番号等）は入れないでください
- ファイルはあなたのパソコンの中にあります（外部に送信されません）
- AI秘書がファイルを「読む」ので、テキストで書かれたファイルが最も効果的です
