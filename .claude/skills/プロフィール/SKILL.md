# プロフィールスキル

## ミッション

「この人に頼みたい」と思わせるプロフィールを書く。
実績の羅列ではなく、読んだ人が「自分のことをわかってくれそう」と感じる文章を作る。

---

## 実行手順

### Step 1: 事前確認

`templates/user-profile.md` を読んでユーザーの事業・ターゲット・実績を把握する。

以下をユーザーに確認する（AskUserQuestion で1問ずつ）:

1. どこで使うプロフィールですか？（Instagram・note・LP・名刺・自己紹介文 等）
2. 読んだ人に何を感じてほしいですか？（「信頼できそう」「親しみやすい」「プロっぽい」等）
3. あなたが今のお仕事を始めたきっかけ・ストーリーを教えてください
4. お客様の変化・実績があれば教えてください（数字があると強い）

user-profile.md に情報がある場合はそちらを優先し、不足分のみ聞く。

### Step 2: 媒体別プロフィールを執筆する

**Instagram プロフィール（150文字以内）:**

```
[何者か（一言で）]
[誰のため（ターゲット）]
[何ができるか（ベネフィット）]
[実績・数字]
[CTA（LINEへ・HPへ）]
```

**note プロフィール・LP用（300-500字）:**

```
[キャッチ一文（共感から入る）]
[過去の自分・きっかけ]
[転機・学び]
[今何をしているか]
[どんな人を助けられるか]
[実績・お客様の声]
[一言メッセージ]
```

**名刺・短い自己紹介（100字以内）:**

```
[役職・肩書き]
[誰の何を解決するか]
[一番の特徴]
```

**文体ルール:**
- 一人称は「私」統一
- 過去のストーリー（before）を入れると共感が生まれる
- 実績は具体的な数字で（「多くの方に」ではなく「200名以上に」）
- ターゲットが「自分のことを書いてくれている」と感じる言葉を使う

### Step 3: 複数パターンを提案する

同じ媒体でも「フォーマル版」「親しみやすい版」の2パターンを用意する。

---

## 禁止パターン

- 実績の羅列だけで終わる（ストーリーを入れる）
- 「〇〇に情熱を持っています」「〇〇が大好きです」だけで実績がない
- 根拠のない「業界No.1」「唯一無二」
- 媒体に合わない文字数（Instagram で500字等）
- 絵文字の使用

---

## 納品チェックリスト（全項目○でなければ出力しない）

- [ ] 媒体に合った文字数
- [ ] ターゲットが読んで「自分のことだ」と感じる言葉が入っている
- [ ] 過去のストーリー（きっかけ・転機）が入っている
- [ ] 実績が具体的な数字で書かれている
- [ ] CTA（次のアクション）が明示されている（LP・note・Instagramの場合）
- [ ] 最低2パターン（フォーマル版・親しみやすい版）提案している
- [ ] 絵文字: なし
