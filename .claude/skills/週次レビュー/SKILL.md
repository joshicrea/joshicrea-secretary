# 振り返りスキル

## ミッション

「今週何をしたか」を振り返り、「来週何に集中すべきか」を明確にする。
使い続けるほどAI秘書があなたのビジネスパターンを把握する。

---

## 実行手順

### Step 1: 事前確認

`{{SECRETARY_BASE_DIR}}/ユーザープロフィール.md` を読んでユーザーの事業・目標を把握する。

Google Calendarが連携されている場合:
`mcp__claude_ai_Google_Calendar__list_events` で今週の予定を取得する。

### Step 2: 今週の振り返りをヒアリング

AskUserQuestion で以下を確認する（1問ずつ）:

1. 今週やったこと・完了したことを教えてください
2. 予定していたけどできなかったことはありますか？
3. 今週「うまくいった」と感じた出来事は？
4. 「難しかった・つまずいた」ことはありますか？

### Step 3: 振り返りの整理と分析

ヒアリング内容を以下の形式でまとめる。

```
【今週の振り返り】（YYYY年M月第N週）

完了したこと:
- 〇〇
- 〇〇

できなかったこと:
- 〇〇（理由: 〇〇）

うまくいったこと:
- 〇〇（再現ポイント: 〇〇）

つまずいたこと:
- 〇〇（原因: 〇〇）
```

### Step 4: 来週の優先事項を提案

振り返りの内容から「来週やるべきこと」を提案する。

優先順位の基準:
1. 期限があるもの（最優先）
2. 今週できなかった重要事項
3. 「うまくいった」の再現・拡張
4. 「つまずき」の根本対策

```
【来週の優先事項】

TOP3（これだけやれば週が成功）:
1. 〇〇（理由: 〇〇）
2. 〇〇（理由: 〇〇）
3. 〇〇（理由: 〇〇）

余力があればやること:
- 〇〇
- 〇〇
```

### Step 5: 学習の記録（自動）

振り返り内容を `{{SECRETARY_BASE_DIR}}/memory/weekly-review/YYYY-MM-DD.md` に保存する。
これが蓄積されるほどAI秘書があなたのパターンを把握できる。

---

## エラーフォールバック

### MCP未接続時
- Google Calendar MCPが未接続の場合: 「今週の予定をここに貼り付けてください」と案内して、貼り付けられた内容をもとに振り返りを進める
- Gmail MCPが未接続でも、このスキルへの影響なし

### ファイル不在時
- `{{SECRETARY_BASE_DIR}}/memory/weekly-review/` が存在しない場合は自動作成してから保存する
- `{{SECRETARY_BASE_DIR}}/ユーザープロフィール.md` が存在しない場合: セットアップが未完了の可能性があるため「AI秘書のセットアップを先に完了してください。「はじめまして」と言えば始められます」と案内する

### ツール実行エラー時
- ファイル保存エラー時: 「振り返りファイルの保存に失敗しました。Claude Codeを再起動してから試してください」と日本語で案内する
- エラー内容をそのまま英語で見せない
