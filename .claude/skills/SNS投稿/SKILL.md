# SNS投稿スキル

## ミッション

スクロールが止まる投稿文を作る。
「いいね」より「保存」「コメント」「フォロー」につながる投稿を目指す。

---

## 実行手順

### Step 1: 事前確認

`templates/user-profile.md` を読んでユーザーのターゲット・トーン・使用SNSを把握する。

以下をユーザーに確認する（AskUserQuestion で1問ずつ）:

1. どのSNS向けですか？ → [Instagram] [X（旧Twitter）] [YouTube概要欄]
2. 伝えたいこと・テーマを教えてください
3. この投稿を見た人に何をしてほしいですか？（フォロー・保存・DM・リンクへ誘導 等）

---

## Instagram 投稿文

### キャプション構成

1. **冒頭1行**: スクロールを止める一文（疑問・数字・共感・衝撃の事実）
2. **本文**: 3-5段落。1段落2-4行
3. **CTA**: フォロー・保存・コメント・URLのどれか1つ
4. **ハッシュタグ**: 20-30個（投稿内容・ターゲット・地域・規模で選ぶ）

**文体ルール:**
- 改行を多用してスマホで読みやすく
- 「あなた」で語りかける
- 専門用語は使わないか使う場合は説明する

**ハッシュタグの選び方:**
- 大タグ（100万件以上）: 5個以内
- 中タグ（1万〜100万件）: 10-15個
- 小タグ（1万件以下）: 残り（ニッチで見つけてもらいやすい）

---

## X（旧Twitter）投稿文

### 基本ルール

- 140文字以内（1ツイート）
- スレッド形式（複数ツイート）の場合は1つ目で続きを読む理由を作る

### 構成パターン

**単発ツイート:**
```
[インパクトのある一文]

[根拠・具体例]

[CTA or 問いかけ]
```

**スレッド（長い内容）:**
```
1/ [結論・一番伝えたいこと]
2/ [理由1]
3/ [理由2]
4/ [理由3]
5/ [まとめ・CTA]
```

---

## YouTube 概要欄

### 構成

```
[動画の要約（2行以内）]

▼ この動画の内容
00:00 〇〇
〇〇:〇〇 〇〇
〇〇:〇〇 〇〇

▼ 関連リンク
[LINE・HP・他の動画 等]

▼ チャンネル登録はこちら
[チャンネルURL]

#キーワード1 #キーワード2 #キーワード3
```

---

## 禁止パターン

- 全媒体で同じ文章を流用する（媒体ごとに最適化する）
- CTAが複数ある（1投稿1CTA）
- 「詳しくはこちら」だけで終わる（何が得られるかを書く）
- 絵文字の使用

---

## 納品チェックリスト（全項目○でなければ出力しない）

- [ ] 冒頭: スクロールが止まる一文になっている
- [ ] CTA: 1つだけ、具体的な行動先あり
- [ ] 媒体に合ったフォーマット
- [ ] Instagram: ハッシュタグ20個以上
- [ ] X: 140文字以内（スレッドの場合は各ツイート140文字以内）
- [ ] 省略なし: 全文書き切り
- [ ] 絵文字: なし
